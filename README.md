# homebrew-tap

Homebrew tap for [Tim Case](https://github.com/timrcase)'s tools.

## Install

```sh
brew install timrcase/tap/claude-code-statusline
```

`brew` expands `timrcase/tap` to this repo (`timrcase/homebrew-tap`) and reads
the formula from [`Formula/`](Formula/).

## Formulae

| Formula | Description |
|---------|-------------|
| [`claude-code-statusline`](https://github.com/timrcase/claude-code-statusline) | Fast, config-driven statusline for Claude Code |

The formulae here are generated and updated automatically by each project's
release workflow — see `scripts/render-formula.sh` in the source repo.
